# bench
Simple electron app for communication services

## Getting started:
```
  npx degit https://github.com/c0z0/bench bench
  cd bench
  yarn install
```

Starting the desktop app:
```
  cd packages/desktop
  yarn electron-deb
```

Starting the website
```
  cd packages/www
  yarn dev
```
